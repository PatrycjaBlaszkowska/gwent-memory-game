Instructions

Gwent memory game is a mix & match game. Player needs to flip two identical cards to get a point.

Take on the challenge to match all the pairs before the time runs out.

You have 60 seconds to match all pairs before a game is over!

Don't worry if you don't. You can still save your score on a leaderboard.

Keep practicing and become a Gwent Memory Game master!
